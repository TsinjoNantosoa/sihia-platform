# Frontend React (implementation progressive)

## Objectif MVP frontend
Offrir une interface claire pour: login, patients, medecins, rendez-vous, dashboard.

## Stack
- React + TypeScript + Vite
- React Query
- Zustand
- TailwindCSS

## Ecrans MVP
1. Login
2. Dashboard
3. Liste patients + fiche patient
4. Liste medecins
5. Calendrier rendez-vous

## Regles UX
- Actions critiques visibles
- Messages d erreur explicites
- Formulaires valides cote client et serveur
- Navigation simple pour personnel non technique

## Sprint UI
- F1: Auth + layout global
- F2: Patients
- F3: Medecins
- F4: Rendez-vous calendrier
- F5: Dashboard KPI

## Definition of done frontend
- Aucun blocage fonctionnel sur parcours principal
- Temps chargement page principale acceptable
- Tests de composants critiques
- Build production sans erreurs
