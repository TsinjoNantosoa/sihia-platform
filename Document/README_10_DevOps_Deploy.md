# DevOps et deploiement

## Objectif MVP
Avoir un deploiement local stable, reproductible et simple.

## Environnements
- local (Docker Compose)
- staging
- production

## CI/CD minimum
1. Lint
2. Tests
3. Build images
4. Push registry
5. Deploy staging manuel

## Observabilite
- Logs centralises
- Metrics API et DB
- Health checks

## Securite DevOps
- Scan image Docker
- Scan dependances
- Rotation secrets

## Definition of done DevOps
- Pipeline CI vert
- Un deploy staging fiable
- Rollback documente
- Health checks passes
